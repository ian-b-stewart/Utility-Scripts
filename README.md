# Utility Scripts

Miscellaneous scripts I use for file cleanup and management.

## Scripts

- **[Find Duplicates](Find_Duplicates/README.md)** (Python) - finds duplicate files across directories using content hashing, spits out a CSV report
- **[Remove Duplicate M4A/WAV](Remove_Duplicate_M4A_WAV/README.md)** (PowerShell) - after converting audio files for Rekordbox importing, cleans up the leftover duplicate formats
